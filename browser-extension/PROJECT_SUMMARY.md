## 🚀 部署指南

### 开发者模式安装
1. 下载 `browser-extension` 文件夹
2. 打开 Chrome 扩展管理页面
3. 启用开发者模式
4. 加载已解压的扩展程序
5. 选择 `browser-extension` 文件夹
