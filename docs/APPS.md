# 应用与脚本列表（Apps）

本仓库包含以下主要应用/脚本：

## 1) QZone Praise Automator（自动点赞脚本）

- 位置：`QZone Praise Automator 2.11.0.js`
- 作用：自动在 QQ 空间对内容进行点赞，支持批量与规则化。
- 环境：Tampermonkey/Violentmonkey，Chromium 内核浏览器优先。
- 安装：将脚本复制到管理器中新建脚本并保存，或通过脚本管理器的“导入”功能加载。
- 结合使用：建议与“QZone Auto Login Helper”配合，在登录过期时自动恢复登录。

## 2) QZone Auto Login Helper（自动登录辅助脚本）

- 位置：`qzone-auto-login-helper/qzone-auto-login-helper.user.js`
- 作用：在登录失效或嵌入跨域登录框时，自动跳转到同源登录页并填入你保存的账号与密码完成登录。
- 控制面板：页面右下角“控制面板”按钮可打开，支持登录方式选择、凭据保存/清除、优先登录页URL、外观设置等。
- 兼容策略：
  - 头像快捷登录：自动查找并点击头像元素。
  - 账户密码登录：支持同源登录页自动填写并提交；出现验证码需手动。
  - 自动填充（已弃用但保留选项）：仅触发基本事件，当两字段已有值时提交。

## 组合使用示例

1. 安装两个脚本到 Tampermonkey。
2. 在登录辅助脚本的控制面板中保存账号与密码；可选配置“优先登录页URL”。
3. 打开 QQ 空间，自动点赞脚本正常运行；若登录过期，登录辅助脚本自动恢复登录，点赞流程不中断。
